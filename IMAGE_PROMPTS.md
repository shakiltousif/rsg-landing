## Quick Generation Tips

### Using AI Image Generators:

**DALL-E 3 (OpenAI):**
- Add: "vector illustration, transparent background, professional design"
- Use: "flat design" or "isometric" for modern look
- Request: "PNG format with transparency"

**Midjourney:**
- Add flags: `--v 6 --ar 3:2 --style raw`
- Use: "vector art, transparent background, clean lines"
- For logos: Add `--no background`

**Figma (Manual Design):**
- Export as PNG with transparent background
- Use vector shapes for scalability
- Apply brand colors (#0ea5e9, #f59e0b, #10b981)

**Adobe Illustrator:**
- Create artboards at specified dimensions
- Use gradient mesh for modern look
- Export: File > Export > Export As > PNG (transparent)

## Image Optimization Checklist

After generating:
1. ✅ Verify transparent background (PNG)
2. ✅ Optimize file size (use TinyPNG or Squoosh)
3. ✅ Keep under 200KB per image (except hero)
4. ✅ Check dimensions match specifications
5. ✅ Test on both light/dark backgrounds
6. ✅ Ensure colors match brand (#0ea5e9, #f59e0b)
